== Installation ==

Ready to go? Here’s how you install Surveys by Feedback Cat in WordPress:

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate Surveys by Feedback Cat through the 'Plugins' menu in WordPress

= Setup of Surveys by Feedback Cat =

Now that it’s installed, here’s how you set up Surveys by Feedback Cat:

1. Go to Surveys by Feedback Cat in your WP Admin menu 
2. In your sidebar, click on 'Surveys by Feedback Cat' to create a new MailChimp form.
3. Build your surveys
4. Set the status to "Active"
5. All submissions from Surveys by Feedback Cat will then be sent straight to your WP admin email

= Troubleshooting Problems with Surveys by Feedback Cat =

What? You mean Surveys by Feedback Cat isn’t perfect?

Yeah, unfortunately. But hey we’re working on it, alright?

So keep calm and carry on ♔.

Because this entire section will help you with any problems you encounter while using Surveys by Feedback Cat.

**Problem: I'm not receiving survey answers via email.**

Solution: Our plugin, Surveys by Feedback Cat, uses the `wp_mail` function - [more info here]([)https://codex.wordpress.org/Function_Reference/wp_mail). 

Depending on your server setup, deliverability of survey results might not always be perfect. 

If that’s the case , here's how you can solve the problem:

1. Check your spam folder for message(s) from Surveys by Feedback Cat.
2. If email(s) from Surveys by Feedback cat is/are NOT in your spam folder, ask you host if sending emails is possible from your server.
3. If you’re still lost, try setting up Mailgun or Mandrill for better email deliverability.
4. Then see if this latter approach works for you with Surveys by Feedback Cat
